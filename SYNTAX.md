## Control Flow Statements
### Conditions
Using an operator from: `==, !=, <, <=, >, >=`. Double statements using `and` and `or`
```nim
variable == true
variabel == false
```
### If Statements

```nim
if condition
{
    //then body;
}
```

```nim
elif condition
{
    //then body;
}
```

```nim
else
{
    //then body;
}
```

### For Loop

```nim
for var name = int to int
{
    //then body;
}
```

```nim
foreach(var item in list)
{
    //then body;
}
```

### While Loop

```nim
while condition
{
    //then body;
}
```
